Det er behov for å tilgjengeliggjøre offentlige dataa på en slik måte er lette å oppdage det går raskt å få vurdert det juridiske grunnlaget for å få tilgang og at det er godt tilrettelagt for å få tilgang til dataene. En nettverksinforastruktur som gir oversik over hvilke data som finnes, hvem som eier dem og hvordan de kan tas i bruk, er nødvendig. ()
