Det gjelder også Altinnorganisasjonen som heller ikke kan være best på alt. I tillegg til å kunne drifte og forvalte Altinnplattformen, og å forstå de grunnleggende fellesutfordringer, bør den være best på å kunne støtte tjenesteeierne i deres arbeid med å tilby Altinnbaserte løsninger på best mulig måte (ref. 4.1.2).
