Melde opphør ()
Lovverket pålegger enhver virksomhetsutøver på eget grunnlag å melde opphør på gitte vilkår. Tradisjonelt med bakgrunn i manglende likviditet og evne til å gjøre opp for seg eller ved et faktisk aktivitetsopphør. Beslutning om oppløsning meldes straks til Foretaksregistret via Samordnet registermelding. Meldingen skal inneholde opplysninger om medlemmene av avviklingsstyret. Husk å legge ved protokollen fra generalforsamlingen. Brønnøysundregistrene kunngjør beslutningen om oppløsning
