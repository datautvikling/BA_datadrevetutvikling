#1 Implementering av CEN 442 standarder med fokus på Data Maler (Data Templates) ()
Gir Ett felles digitalt språk i design - bygging og drift Å ha omforente Data Templates er en forutsetning for vi blir digitale, dvs at maskiner forstår hva som menes. Bør samkjøres nordisk og på europeisk plan Effekt: For å bygge billigere, grønnere, raskere med mindre feil Nytte for Byggherrer, myndigheter, rådgivere, arkitekter, utførende, grossister, produsenter og software aktører Eiet av en norsk uavhengig entitet Forslag: Det settes opp et selskap som opererer uavhengig av aktører, slik at samfunnet og aktørene kan bruke Data Templates man kan stole på. Finansiiering: De som ønsker å bruke Data Templates betaler, på lik linje med standarder, GS1 produkt identifikatorer etc. Eksisterende software må ta i bruk Data Templates og properties, og ikke finne opp dette selv.
