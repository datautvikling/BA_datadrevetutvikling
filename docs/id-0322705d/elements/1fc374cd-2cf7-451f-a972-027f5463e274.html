Det vil være hensiktsmessig at informasjonen samles ett sted (forvaltningsorgan) som så kan sette sammen ulike "informasjonspakker". Innkjøpere og forbrukere må også forstå informasjonen som blir gitt, og dette bør ikke være en oppgave for det offentlige. Profesjonelle aktører bør bidra med vurderinger og tilrå eller advare mot bruk av leverandører (lyssignal, smilefjes ol) ()
