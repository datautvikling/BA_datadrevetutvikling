Skaffe kapital / investorer (01.3) ()
Are there any tools which help entrepreneurs in gaining financial information / possibilities when starting a business Foutsetter også valg av juridisk form - Banklån - krever forretningsplan og ofte sikkerhet i selskapets eller etablerers personlige eiendeler - Tilskudd eller finansieringsordninger gjennom for eksempel Innovasjon Norge, NAV, etc - etablererstipend hos Innovasjon Norge - Private investorer (egenkapital) - Eksterne investorer krever som regel forretningsplan og eierandeler/ avkastning for å investere i selskapet Tilgangen på kapital er en tilhørende utfordring. For å få banklån eller lån av andre finansinstitusjoner må man som regel stille sikkerhet i eiendeler eller lignende. Man kan ha godt bearbeidede forretningsplaner der det til og med er inngått avtaler med leverandører og kunder, og banken uttrykker seg positivt til planene, men uten sikkerhet i for eksempel etablererens bolig eller tilsvarende eiendeler så kan de ikke tilby lån.
