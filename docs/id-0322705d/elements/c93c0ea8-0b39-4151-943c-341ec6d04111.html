21. Det offentlige bør betydelig styrke og målrette høyere utdanning innen IKT-fag. ()
Spesielt rettet mot sentrale områder som dataanalyse (data science), dataingeniør (dataengineering) og kunstig intelligens, men også innen fag som (data) sikkerhet, produktledelse (product management) og brukeropplevelse (UX development/tjenestedesign), samt å styrke IKT- forståelsen i juridiske og økonomiske fag.
