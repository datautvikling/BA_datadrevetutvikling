SKE-Innkreving: Bedrifter opplever informasjon i dag for generell og har behov for tilrettelagt veiledning for deres type bedrift, som også tar hensyn til situasjonen bedriften er i. Tilrettelagt veiledning baserer seg på å samle all data relatert til bedriften for å forstå situasjonen de er i og gi konkret veiledning ut fra det. Veiledningen bygges inn i tjenester og kontaktpunkter brukeren benytter for proaktiv og relevant veiledning der og da. Ved behov kan også veiledning utføres som rådgivingssamtale - med saksbehandler i dag og en robot frem i tid? Hendelser som påvirker brukers situasjon fanges ikke opp/deles ikke i det offentlige, slik at eventuell veiledning blir mangelfull og Skatteetaten har heller ingen tradisjon for å ta initiativ til proaktiv hjelp. Det medfører at problemene for virksomheten blir store før de fanges opp og riktige grep kan gjennomføres. Det gjelder blant annet ENK som ikke avsluttes tidsnok eller på riktig måte, og små bedrifter som ikke leverer mva-meldingen tidsnok og dagsbøtene (tvangsmulkt) blir starten på en dårlig spiral.
