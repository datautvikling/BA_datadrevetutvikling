Seriøsitetsdata (næringslivsinformasjon)
Målbilde for deling av seriøsitetsinformasjon på nasjonalt nivå, på tvers av opplysningskilder og brukergrupper
Juridisk: Forvalter/forvaltningsorganet for datadeleren/eBevis må få egen lovhjemmel til å innhente, sammenstille og videreformidle seriøsitetsopplysninger fra ulike kilder.
Organisatorisk: Noen må vurdere hvilke opplysninger de ulike konsumentgruppene behøver og har lov til å få.
Semantisk: Opplysningene fra de ulike virksomhetene som kan si noe om seriøsitet er ikke umiddelbart enkle å tolke, og ofte må de ses i sammenheng med andre opplysninger for at man skal kunne utlede noe om hvorvidt en virksomhet er seriøs eller ei.
Teknisk: Teknisk sett er det meste på plass allerede hos Skatteetaten og Altinn. SKE jobber i disse dager med å tilgjengeliggjøre langt flere opplysninger for eBevis enn hva som gjøres i dag (i dag er det kun restanser).

Det vil være hensiktsmessig at informasjonen samles ett sted (forvaltningsorgan) som så kan sette sammen ulike "informasjonspakker". Innkjøpere og forbrukere må også forstå informasjonen som blir gitt, og dette bør ikke være en oppgave for det offentlige. Profesjonelle aktører bør bidra med vurderinger og tilrå eller advare mot bruk av leverandører (lyssignal, smilefjes ol)
Skatteetaten skal kun levere egne data til eBevis, eventuelle opplysninger fra andre offentlige myndigheter skal ikke leveres via Skatteetaten.
